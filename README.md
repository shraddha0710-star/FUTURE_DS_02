# FUTURE_DS_02

## Customer Retention & Churn Analysis Dashboard

This project focuses on analyzing customer churn and retention patterns using Power BI. The dashboard provides valuable business insights through interactive charts and KPI cards.

## Objectives
- Identify churned customers
- Analyze retention trends
- Understand customer behavior
- Improve business decision-making

## Tools Used
- Power BI
- Excel
- Data Cleaning
- Data Visualization

## Dashboard Features
- KPI Cards
- Churn by Gender
- Tenure Analysis
- Plan Type Churn
- Revenue Insights
- Customer Trends

## Key Insights
- Customers with shorter tenure show higher churn.
- Some plan types have more churn than others.
- Retained customers generate stable revenue.
- Support quality can impact customer retention.

## Conclusion
This dashboard helps businesses reduce churn, improve retention strategies, and increase long-term customer value.
